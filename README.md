# Simple Interest Calculator

Một ứng dụng web đơn giản giúp tính toán lãi đơn dựa trên số tiền gốc, lãi suất và thời gian.

## Công thức tính

Ứng dụng sử dụng công thức toán học cơ bản để tính lãi đơn:
$$I = P \times r \times t$$

Trong đó:
*   **$I$** (Interest): Tổng tiền lãi
*   **$P$** (Principal): Số tiền gốc ban đầu
*   **$r$** (Rate): Lãi suất hàng năm (tính theo phần trăm)
*   **$t$** (Time): Thời gian vay hoặc gửi (tính theo năm)

## Công nghệ sử dụng
*   **HTML5:** Xây dựng cấu trúc giao diện.
*   **CSS3:** Làm đẹp và tạo bố cục thân thiện với người dùng.
*   **JavaScript:** Xử lý logic tính toán trực tiếp trên trình duyệt.

## Hướng dẫn sử dụng
1. Tải toàn bộ mã nguồn về máy hoặc clone repository này.
2. Mở file `index.html` bằng bất kỳ trình duyệt web nào (Chrome, Edge, Safari,...).
3. Nhập số tiền gốc, mức lãi suất và số năm.
4. Nhấn nút **"Calculate"** để xem tổng tiền lãi và tổng số tiền thu về.
